Get Facebook access token for BeeActive API

Click the button below, sign in with Facebook, then either Send to API (creates/finds user in DB and returns your app JWT) or copy the token for Insomnia: POST http://localhost:3000/auth/facebook with body {"accessToken": "<token>"}

1) In this file, set FB_APP_ID to your Facebook App ID (Meta for Developers → Your app → Settings → Basic). 2) In the Facebook app: Products → Facebook Login → Settings → add http://localhost:4200 to Valid OAuth Redirect URIs. 3) Serve: npx serve -p 4200 . then open http://localhost:4200/facebook-token-test.html
