Get Google ID token for BeeActive API

Click the button below, sign in with Google, then either Send to API (creates/finds user in DB and returns your app JWT) or copy the token for Insomnia: POST http://localhost:3000/auth/google with body {"idToken": "<token>"}

This page must be served from http://localhost:4200 (run: npx serve -p 4200 . from project root).
